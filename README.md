This project is a GUI-based Ordering System called “Angels Burger: Burger ng Bayan Ordering System” created using the Java Programming Language in Apache NetBeans. I developed this project as a first year student during the 2nd semester at Cristal E College.

The purpose of this system is to provide a simple and user-friendly ordering process for customers. Through this project, I learned the basics of Java GUI development, event handling, and system design while improving my programming skills as an IT student.
